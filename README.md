# 📊 Blinkit Sales Dashboard – Power BI Capstone Project

Blinkit Sales Dashboard is an end-to-end Business Intelligence project built using Power BI to analyze sales performance, customer trends, and product insights through interactive visualizations.

---

## 🚀 Tech Stack

* Power BI Desktop
* Power Query (Data Cleaning & Transformation)
* DAX (Data Analysis Expressions)
* Excel/CSV Dataset

---

## 🧹 Data Cleaning

* Removed duplicate and null values
* Handled missing data appropriately
* Standardized column names and formats
* Transformed raw data using Power Query
* Created calculated columns and measures

---

## 🌟 Features

✅ Interactive Dashboard
✅ KPI Tracking (Sales, Profit, Orders)
✅ Sales Trend Analysis
✅ Product & Category Performance
✅ Regional Insights

---

## 📊 Key Insights

* Identified top-performing products and categories
* Analyzed sales trends over time
* Highlighted high and low performing regions
* Provided insights for business decision-making
